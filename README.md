# RustCrypt
x86-64 Crypter built in Rust for Windows with Anti-VM, powered by memexec

## Usage

### Supported targets
- Windows x86-64
- Windows x86

### Limitations
- .NET not supported
- Files over 600MB

## Disclaimer
This is a tool used to test the Dynamic detection capabilites of AV and EDR, use of this project is on your own risk
